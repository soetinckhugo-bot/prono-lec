## Règle de normalisation

Le nom de fichier est automatiquement généré à partir du nom de l'équipe :

1. Suppression des accents
2. Passage en minuscules
3. Suppression de tout ce qui n'est pas lettre ou chiffre

Exemple : `Karmine Corp` → `karminecorp.png`

Si tu ajoutes une nouvelle équipe, place son logo PNG ici avec le nom normalisé.
